# Safe Bilhar POS Pro

Sistema POS profissional em React + Tailwind, inspirado em sistemas modernos de venda como Loyverse POS.

## Inclui

- Login demo
- Dashboard com gráficos
- POS com carrinho
- Produtos e stock
- Clientes
- Histórico de vendas
- Relatórios
- Configurações da loja
- Layout responsivo para computador, tablet e celular
- Preparado para Firebase
- Pronto para GitHub e Vercel

## Como rodar no computador

```bash
npm install
npm run dev
```

Depois abra o endereço que aparecer no terminal.

## Como enviar para GitHub

1. Extraia este ZIP.
2. Envie todos os arquivos e pastas para o repositório.
3. No Vercel, importe o repositório.
4. Framework: Vite.
5. Build command: `npm run build`
6. Output directory: `dist`
